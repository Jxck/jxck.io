Intro

2019/1/13(日) に、以下で告知した「次世代 Web カンファレンス」を開催した。

次世代 Web カンファレンス 2019 開催告知 | blog.jxck.io

前日に初雪が観測されて心配したが、天気にも恵まれ、開催趣旨の通り予定していたセッションを全て終えることができた。

次世代 Web カンファレンス - connpass
