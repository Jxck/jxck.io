第 73 回のテーマは Semantic Web です。

今回は セマンティック・ウェブのためのRDF/OWL入門 や セマンティック HTML/XHTML の著者であり、 ジャパンサーチ などを手掛ける Semantic Web の専門家、 神崎正英 先生をお迎えし Web におけるセマンティクスの歴史や RDF の変遷、ジャパンサーチで行われている作業から見た Semantic Web の現状などについてお聞きしつつ、これからの Web のセマンティクスなどについて議論させていただきました。
