# ep43 Permission

## Theme

第 43 回のテーマは Permission です。

今回は [@toyoshim](https://twitter.com/toyoshim) をお呼びし、今議論中の Web における Permission について、何が問題/課題であり、現状どうなっているのか。

実装と仕様の両方の側面から議論し、 Web における Permission の扱いが Web をどのように変えていくのかについて議論しました。
