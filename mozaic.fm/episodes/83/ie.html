Theme

第 83 回のテーマは IE です。

今回は Microsoft の Yusuke さんをお呼びして、 IE のたどってきた歴史を振り返り、これをどう Edge に置き換えていくのかという流れを踏まえながら、 IE が Web に与えた歴史や、 IE 以降の Web について議論しました。
